# Simple-pixel-monster-hunting-game
A simple pixel monster-hunting game built with C and EasyX. Covers basic C, data structures, and EasyX concepts—great for beginners! Not perfect, just for learning. I'm also a newbie, so feel free to discuss or ask questions. Thanks!

#Features
Player Movement and Controls
Monster Spawning and AI Movement
Collision Detection and Damage System
Basic Game Loop and Rendering

#Environment Dependencies
Windows Operating System
EasyX Graphics Library 
C Language Compiler (Visual Studio )

Since this game relies on a lot of textures, and I haven't quite figured out how to properly upload them to GitHub yet, this code won't run. Please just use it as a reference
